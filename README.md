# Portfólio pessoal
O objetivo principal desse projeto é de centralizar informações pessoais e disponibilizar vídeos e/ou imagens de projetos pessoais, acadêmicos e profissionais.

#### Status do projeto: Em andamento.

## Desenvolvedor
Lucas Diniz

## Tecnologias
- PHP
- HTML
- CSS
- JavaScript
- JSON


## Telas e Funções
### Sobre
- [ ] Apresentação. Texto contendo informações gerais sobre mim.

### Conhecimentos
- [ ] Lista completa de cursos.
    - Nome do curso.
    - Descrição.
    - Data de conclusão ou status do curso.
    - Espaço para que possa acrescentar ferramentas relacionadas ou conhecimentos extras.

### Projetos
- [ ] Cartões com projetos.
    - Título do projeto.
    - Área de mídia (1 vídeo, 2 imagens e um botão de "Abrir galeria" ou "+X").
    - Breve descrição.

### Info. Serviços
- [ ] Informações sobre serviços.
    - Serviços realizados.
    - Descrições detalhadas sobre pagamentos dos serviços realizados.
    - Contato e formas de pagamento.

### Galeria
- [ ] Área para exibição completa do projeto. Nome, categoria, descrição, vídeo e imagens.
    - Será exibida apenas ao clicar em "Abrir galeria/+X".
